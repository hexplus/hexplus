### 👋 Hey, I'm Jose Ramírez

I'm an **AI Full Stack Developer** based in 🇨🇷 Costa Rica, passionate about building lean, fast, and developer-friendly tools. I created **[SibuJS](https://github.com/hexplus/sibujs)** — a zero-dependency, signal-based reactive framework with no Virtual DOM overhead.

- 🔭 Currently building the **SibuJS ecosystem** — framework, UI library & CLI
- 🤖 Working at the intersection of **AI and frontend development**
- 🧠 Interested in reactive systems, low-level performance and clean abstractions
- 🏠 Working remotely from Costa Rica
